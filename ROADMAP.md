# Roadmap — MoA Engine

## v1.1 — Производительность

- [ ] **Параллельный запуск пропозеров** с token-bucket rate limiter вместо последовательного (ускорение в 3×)
- [ ] **Streaming ответа** через MCP (токены по мере генерации)
- [ ] **Поддержка Groq** — Mixtral, Llama 3 (бесплатно, быстрее OpenRouter)
- [ ] **Поддержка DeepSeek** — DeepSeek V3 (бесплатно, 1M контекст)
- [ ] **Динамический выбор моделей** — если модель падает с 429, автоматически переключаться на запасную

## v1.2 — Качество

- [ ] **Включить quality scoring** (`quality.enabled: true`) — coherence через judge model
- [ ] **Self-critique** — финальная модель проверяет и улучшает свой ответ
- [ ] **RAG** — добавить контекст из файлов/URL в пропозеры
- [ ] **Multi-turn** — поддержка диалога (история сообщений)
- [ ] **Выбор top-K** — динамический выбор лучших ответов Layer 1 по качеству

## v1.3 — UX & Интеграции

- [ ] **Telegram бот** — `/moa "запрос"` через python-telegram-bot
- [ ] **Discord бот** — через discord.py
- [ ] **API сервер** — FastAPI эндпоинт `/api/moa` для внешних вызовов
- [ ] **Web UI улучшения** — streaming, история, экспорт в Markdown
- [ ] **Docker** — `Dockerfile` + `docker-compose.yml` для одно-командного запуска

## v1.4 — Расширение скилов

- [ ] **Свой репозиторий скилов** — создать `Davinchi21/moa-skills` с авторскими скилами
- [ ] **Валидатор скилов** — проверять SKILL.md на корректность (YAML frontmatter)
- [ ] **Авто-обновление** — `skills update --all` скачивает новые версии с GitHub
- [ ] **Категории скилов** — группировка по тегам (k8s, cloud, terraform...)

## v2.0 — Advanced

- [ ] **MoA-as-a-Service** — облачный сервер с веб-интерфейсом
- [ ] **Кастомные модели** — поддержка локальных LLM через Ollama/vLLM
- [ ] **Fine-tuning** — дообучение аггрегатора на парах (вопрос → лучший ответ)
- [ ] **Экспорт в OpenRouter** — поделиться конфигом как публичным preset
- [ ] **Бенчмаркинг** — автоматическое тестирование качества разных комбинаций моделей
